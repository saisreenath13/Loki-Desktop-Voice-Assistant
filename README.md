**Loki** is a next-generation Desktop Voice Assistant that bridges the gap between system automation and Generative AI. Unlike traditional assistants that only perform basic tasks, Loki uses the **Google Gemini Pro API** to understand natural language, answer complex queries, and converse intelligently, while retaining the ability to control your PC.

Wrapped in a beautiful, futuristic **Dark Mode GUI**, Loki is designed to be your always-on desktop companion.
